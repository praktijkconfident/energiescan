# Tool Starter

Starter template voor de workshop **Bouw jouw eigen Interactieve Kennis Tool** van [Energieke Lieke](https://energiekelieke.nl).

## Aan de slag

Volg het interactieve stappenplan in jouw Kennis Shop leeromgeving.

## Structuur

```
app/
  globals.css       ← jouw kleuren en lettertype
  tool/
    page.tsx        ← jouw artifact-code
    layout.tsx      ← iframe-ready (geen navigatie)
public/
  fonts/            ← upload hier jouw eigen fontbestanden (optie B)
```

## Lokaal draaien

```bash
npm install
npm run dev
```

Open daarna [localhost:3000/tool](http://localhost:3000/tool).
